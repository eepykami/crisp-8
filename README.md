# crisp-8
 CHIP-8 interpreter written in C. Work in progress. Currently only supports only the instructions needed to load the IBM ROM.

## Usage
```bash
crisp-8 <path-to-ROM>
```

## Build

Clone this repo including submodules, then:

```bash
mkdir build
cd build
cmake ..
make
```
