# crisp-8
 CHIP-8 interpreter written in C. Work in progress.

## Usage
```bash
crisp-8 <path-to-ROM>
```

## Build
### Build dependencies
```bash
sudo apt install libsdl2-dev
```

Clone this repo, then:

```bash
mkdir build
cd build
cmake ..
make
```
